## What is this?

Innovator Crews is a premium web design & digital services agency based in the Philippines, serving clients worldwide. This repository contains the agency's high-performance single-page marketing website, built without frameworks to showcase the team's services, portfolio, and process. The single most important thing to know: every line of code is handcrafted vanilla HTML, CSS, and JavaScript.

## Key Features

- **Sticky Navbar** — Scroll-aware navigation with hamburger menu, shadow detection, and active-section tracking.
- **Animated Hero** — Blob shapes and mesh gradient backgrounds create a dynamic first impression.
- **Browser Mockup** — Interactive browser window mockup with floating stat cards (+300% Traffic, 5.0/5.0 Satisfaction).
- **Animated Counter Strip** — Numbers count up on scroll (20+ projects, 3+ years, 100% satisfaction, 24h response).
- **Services Bento Grid** — Six services displayed in a modern bento-style card layout (Website Design, E-Commerce, Brand & UI/UX, SEO, Maintenance & Support).
- **Portfolio Filter** — Project gallery with category filters (All, Platform, System, E-Commerce) linking to five live deployments.
- **Scroll-Triggered Animations** — Elements animate in as they enter the viewport via IntersectionObserver.
- **Testimonials Carousel** — Rotating client quotes with 5-star ratings.
- **Process Walkthrough** — Visual 4-step guide: Discovery → Proposal → Design → Build & Launch.
- **Fully Mobile-Responsive** — No build tools or frameworks; runs entirely in the browser on any device.
